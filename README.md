## <img src="img/CypressD.png" width="120" height="60" align=center>  
## How to use:
### Run all tests locally with UI:
```
yarn cy:open
```
---
###Run all tests locally without UI:

```
yarn cy:run
```
⚠️ *Tests will be run on Google Chrome*

```

```

---

### Clear Allure report:
```
yarn allure:clear
```
